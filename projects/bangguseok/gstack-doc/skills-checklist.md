# 방구석 사진관(PassPro AI) - Gstack 스킬 활용 체크리스트

프로젝트의 성공적인 마무리와 배포 후 안정적 운영을 위한 Gstack 스킬 체크리스트입니다. 완료된 스킬과 앞으로 진행할 스킬을 표 형태로 정리하여 추적 관리가 용이하도록 구성했습니다.

## 1. 완료된 핵심 스킬 (Completed Core Skills)
| 진행 상태 | 스킬명 | 목적 및 세부 설명 |
| :---: | :--- | :--- |
| `[x]` | `/office-hours`<br>`/plan-ceo-review` | 비즈니스 모델 발굴, 방향성 설정, "10-star" 프로덕트 핵심 세일즈 포인트 복원 및 카피라이팅 확립. |
| `[x]` | `/plan-eng-review` | 백엔드 데이터 플로우 구조화 및 인프라 아키텍처(Gemini 연동 등) 설계 확립. |
| `[x]` | `/design-consultation`<br>`/plan-design-review` | 전반적인 컨셉("햇빛 쏟아지는 방구석 사진관") 방향성에 맞는 프리미엄 디자인 시스템(색상, 폰트, 호버 애니메이션) 구축. |
| `[x]` | `/qa` | 브라우저 가동을 통한 E2E 로직(이미지 업로드 → 검사 → 다운로드) 전체 통합 테스트 진행 및 오류 핫픽스 완료. |
| `[x]` | `/cso` (Security) | OWASP 기반 악성 파일 무력화 조치. IP 기반 Rate Limit(분당 5회) 및 Byte 단위 파일 서명 검사(Magic Number) 구현. |
| `[x]` | `/review` | PR 사전 리뷰. N+1 문제, Race Condition, 성능 최적화, 컴포넌트 마운트 구조 등 Staff 엔지니어 수준의 코드 리뷰 완료. |

## 2. 완료했지만 추후 재가동을 추천하는 스킬 (Replayable Skills)
| 진행 상태 | 스킬명 | 재가동 시점 및 목적 |
| :---: | :--- | :--- |
| `[ ]` | `/design-review` | **배포 직전 혹은 신규 페이지 추가 시점:**<br>CSS 애니메이션, 반응형 레이아웃 뷰, 호버 등 시각적 결함을 실제 기기 단위에서 한 번 더 크로스 오버 체크합니다. |
| `[ ]` | `/qa-only` (또는 `/qa`) | **향후 결제 모듈(Toss, Stripe) 연동 시점:**<br>코드를 직접 수정하지 않고 봇이 브라우저만 제어하며 E2E 플로우 중 실패 지점을 Report 전용으로 제공받기 위해 가동합니다. |
| `[ ]` | `/debug` | **알 수 없는 Production 에러 발생 시:**<br>원인 모를 파일 업로드 실패, 네트워크 Timeout 시 코드 수정 전 Root Cause를 찾기 위해 심층 분석을 요구할 때 즉시 투입합니다. |
