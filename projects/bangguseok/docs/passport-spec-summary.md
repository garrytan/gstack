# 한국 여권 사진 규격 가이드

## 출처
- 외교부 여권안내 (2018년 1월 개정)
- `passport_image_guide.pdf` 원본 참고

## 규격

### 사진 크기
| 항목 | 값 |
|------|-----|
| 물리 크기 | 3.5cm × 4.5cm |
| 온라인 규격 | 413px × 531px (~300dpi) |
| 파일 형식 | JPG / JPEG |
| 파일 크기 | 500KB 이하 |
| 유효 기간 | 촬영 후 6개월 이내 |

### 얼굴 비율
| 항목 | 값 |
|------|-----|
| 머리 길이 (정수리~턱) | 3.2cm ~ 3.6cm |
| 세로 비율 | 약 71% ~ 80% |

### 배경 & 조명
- 배경: **순백색 (#FFFFFF)**
- 배경에 그림자, 패턴, 기타 물체 없어야 함
- 얼굴/피부에 조명 반사 없어야 함
- 선명하고 초점이 맞는 이미지

### 얼굴 & 표정
- **정면** 응시 (측면/기울임 불가)
- 자연스러운 **무표정** (입 다물기)
- 눈은 카메라 직시, 적목현상 없어야 함

### 안경
- 안경 프레임이 눈을 가리면 안 됨
- 렌즈 반사 없어야 함
- 색안경, 선글라스, 컬러 렌즈 불가

### 복장 & 액세서리
- 흰 옷 피하기 (배경과 구분 안 됨)
- 모자, 캡, 이어폰 불가
- 종교적 의상은 매일 착용 시 허용 (얼굴 전체 노출 조건)

### 불합격 사유
- 과도한 포토샵/보정 (기본 밝기·색감 보정은 가능)
- 측면 얼굴, 기울어진 머리
- 입 벌림, 이 보임 (24개월 미만 영아 예외)
- 배경에 그림자 또는 다른 사람/물체
- 흰색이 아닌 배경
