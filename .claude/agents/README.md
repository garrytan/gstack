# The Agency — AI Specialists for Claude Code

144+ specialized AI agent personas organized into functional divisions.
Source: [msitarzewski/agency-agents](https://github.com/msitarzewski/agency-agents)

## Quick Install

```bash
./scripts/install-agency-agents.sh
```

This clones the source repo and copies all agents to this directory.

## Usage

In any Claude Code session, reference agents by name:

```
Activate Frontend Developer and help me build a React component.
Use the Reality Checker to verify this feature is production-ready.
Use the Security Engineer to audit this authentication flow.
Use the Backend Architect to design the database schema.
```
