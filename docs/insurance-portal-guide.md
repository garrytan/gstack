# AMMOR Insurance Portal Guide

## Workflow expectations

- Fast intake and evidence capture.
- Accurate fraud score visibility for assessors.
- Clear explainability for appeal and escalation paths.

## Required operational controls

- Case ownership and handoff rules are explicit.
- AI scores must include context and confidence assumptions.
- Human review path should be explicit and auditable.

## Testing focus

- Claims creation and case state transitions.
- Evidence upload and document retrieval.
- Dashboard usability for adjuster workflow.
