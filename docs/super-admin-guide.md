# AMMOR Super Admin Guide

## Responsibilities

- Configure user roles and access policies.
- Review platform risk and compliance reports.
- Approve high-risk security and deployment exceptions.

## Non-negotiable protections

- Never create super-admin privilege bypasses.
- Never delete super admin account.
- Never alter authentication providers without explicit approval.

## Daily checks

- Review compliance gate outcomes.
- Review unresolved risk findings.
- Confirm deployment rollback plan readiness.
